Welcome to Flask-PynamoDB's documentation!
==========================================

A flask extension that provides integration with PynamoDB.


Features
---------
- Python 3 Support.
- Helper functions for Flask, such as ``get_or_404`` and ``first_or_404``.
- Fully Tested.


Installation
-------------

You can easily install Flask-PynamoDB with **pip**:

.. code-block:: bash

   python -m pip install flask-pynamodb
